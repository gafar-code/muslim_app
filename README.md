
# MUSLIM APP
- slicing design dari figma ke flutter,
- consume API
- state-management menggunakan Getx
- WebView

## Instalasi
```ruby
git clone http://github.com/gafar-code/muslim_app
cd kosan_app
flutter pub get
flutter run
```
## Demo Aplikasi

## API

- Surah Alquran : https://equran.id/api/surat
